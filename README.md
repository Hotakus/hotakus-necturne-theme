## ✨ 特性

- 低对比度暗色，长时间编码不刺眼
- 17 色分层色板 — 变量、类型、字符串等各有独立色系
- 策略性斜体 — 声明关键字斜体，流程控制直立
- Git 低存在感配色 — SCM 和 Diff 视图低透明度

## 📦 安装

```bash
code --install-extension hotakus.hotakus
```

或 `Ctrl+Shift+X` 搜索 **Hotakus's Nocturne**。
